JavaScript tiene, como misión, escribir código que sea lo más compacto posible, que use la menor cantidad de líneas de código posibles, y que nos ofrezca la mayor versatilidad y funcionalidad posible. Existen varios estilos para escribir código en jQuery. A lo largo de esta publicación estoy utilizando un estilo de programación consistente en desglosar, dentro de lo posible, cada una de las líneas de código, e intentar realizar sangrados para que la legibilidad del código resulte lo más agradable posible. Sin embargo, cuando consultemos a lo largo de Internet, en muchas ocasiones podremos ver código en el formato que se presenta en este ejercicio. Este ejercicio contiene exactamente el mismo código que presentamos en el ejercicio anterior, con la diferencia de que, en lugar de estar desarrollado en múltiples líneas, está todo introducido dentro de una misma línea. Si nos fijamos es, exactamente, el mismo código, dentro de que la única que diferencia consiste en que se han eliminado los saltos de línea. Evidentemente no se recomienda utilizar este estilo hasta no tener un nivel bastante alto de familiaridad con jQuery, pero a la vez, en el caso de que, al consultar en diferentes recursos, como por ejemplo Internet, el lector llegue a un punto en el que encuentre código de en este estilo, simplemente deberá tener en cuenta qué es el mismo código que estamos trabajando a lo largo de este curso, la única diferencia es que se encuentra compactado.
