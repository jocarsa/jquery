cualquier etiqueta HTML dispone de una serie de atributos, tanto atributos estándar, como atributos personalizados que nosotros mismos podemos crear. Para este ejemplo, tenemos un código en el cual, en la parte de HTML, tenemos cuatro botones, donde, en cada uno de ellos, he introducido un atributo personalizado, llamado mi color, y para cada uno de ellos, ese atributo tiene un valor diferente, respectivamente, rojo, verde, azul, y naranja. A continuación tengo un divisor, con la identificación contenedor, que es el que va a recibir el resultado de la acción. Podríamos resolver este ejercicio creando un escuchador de Eventos para cada uno de los botones, pero, sin embargo, va a resultar mucho más eficiente que cree un único escuchador genérico para la etiqueta del botón. Permite crear escucha dores de forma automática para cada uno de los cuatro botones, sin tener que declarar código para cada uno de ellos. Por lo tanto, cada vez que, genéricamente, pulsar un botón, se ejecutará el código JavaScript que existe en la segunda parte del ejercicio. Ese código, en primer lugar, detecta el atributo, mi color presente en el botón sobre el cual se ha hecho clic, y lo guarda dentro de una variable. A continuación, se utiliza esa variable para establecer el color de fondo del divisor con el identificador contenedor. Por lo tanto, vemos como, de esta forma, utilizando una cantidad relativamente pequeña de código Fuente, hemos conseguido cambiar el color de un elemento, sin tener que especificar un evento de clic para cada uno de los botones.
