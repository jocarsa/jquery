Y hasta el momento hemos visto que podemos ocultar y podemos mostrar elementos previamente existentes dentro de HTML utilizando jQuery. En este bloque de contenidos vamos a comprobar, como también podemos leer el texto que exista dentro de un selector, así como escribir texto dentro de ese mismo selector. En el caso de que queramos seleccionar únicamente el texto que exista dentro de ese selector, descartando cualquier contenido basado en etiquetas HTML, es decir, convirtiendo ese contenido a texto plano, en ese caso podemos utilizar un método llamado texto. Al igual que ocurre con la gran mayoría de métodos y defunciones en jQuery, el mismo método nos sirve tanto para leer como para escribir. Si en los paréntesis del método no existe ningún contenido, en ese caso se utiliza para leer, y si dentro de esos paréntesis existe algo de contenido, como en este caso, una cadena de caracteres alpha numéricos, en ese caso el método se utiliza para leer. Es por esto que, en el ejemplo que se presenta a continuación, se muestra dos bloques de código, en los cuales, en primer lugar, tenemos un botón de leer que lanza una acción que introduce, dentro de una variable llamada contenido, el texto que exista dentro del divisor que tenga el identificador rojo. Y por último, se muestra ese contenido dentro de la consola. Por otra parte, tenemos también un botón con identificador escribir, de forma que, si lo pulsamos, dentro del divisor con el identificador rojo, se escribirá un texto realizado desde JavaScript
