hasta el momento, en los ejemplos que hemos desarrollado anteriormente, hemos hecho referencia a la pulsación de botones del ratón, cuando lo que se pretende es hacer, de algún modo, clic encima de algún selector dentro de HTML. Hasta el momento, los ejemplos mostrados han tenido como protagonistas a los botones, ya que los botones, desde el punto de vista de la usabilidad, son elementos muy susceptibles de recibir Klixx por parte del usuario. Otro de los elementos de interacción preferente dentro del lenguaje HTML son los campos de los formularios. Existen diferentes tipos de campos, pero para este ejemplo hemos utilizado un clásico campo de entrada de texto correspondiente a un formulario HTML. Dentro de este tipo de elementos, como en este caso son los campos, veremos, en primer lugar, que existen dos eventos importantes relacionados con el ratón. El primer evento importante existe cuando el cursor del ratón no solo pasa por encima del elemento, sino que además, estando el cursor dentro del elemento, el usuario realiza clic para situar el cursor dentro de ese campo, y a continuación, poder empezar a escribir. Este evento se conoce con el nombre de enfoque, o Focus. A continuación, el segundo evento consiste en cuando el usuario retira el foco con respecto al control de formulario, y para ello, mueve el cursor del ratón, fuera del campo, y a continuación hace clic, un evento mediante el cual entendemos que el usuario ha retirado el foco con respecto al control activo del formulario. Esa acción se conoce como desenfoque, o, en inglés, blog.
