en ocasiones, poner propiedades y atributos CSS, dentro de un selector, de forma individual, puede resultar conveniente. Pero cuando sobre un selector queremos, o bien poner, o bien quitar grupos de atributos, en ese caso, probablemente seremos mucho más productivos, si asociamos todo ese grupo de atributos dentro de una clase, y simplemente utilizamos una única instrucción de asignar o quitar una clase de forma dinámica a un elemento. De esta forma, el jQuery encontramos dos métodos, que se presentan en el ejercicio a continuación, mediante los cuales, de forma dinámica, podemos, o bien añadir una clase CSS previamente definida en la hoja de estilo, o bien podemos quitar una clase que previamente haya sido asignada, de forma estática, o de forma dinámica. En este ejercicio que se muestra a continuación, podemos encontrar la aplicación individual de los métodos de añadir clase y de quitar clase a un selector HTML determinado.
